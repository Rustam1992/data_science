# Data Science Projects

В данном репозитории собраны проекты по анализу данных и машинному обучению, выполненные с использованием Python.  
Проекты охватывают ключевые направления Data Science: EDA, статистический анализ, классическое машинное обучение, временные ряды, NLP, Computer Vision и построение ML-пайплайнов.

| Наименование проекта | Описание проекта | Используемые технологии |
|----------------------|------------------|-------------------------|
| Exploratory Data Analysis (EDA) | Проведён разведочный анализ данных: анализ распределений, пропусков, выбросов и взаимосвязей между признаками. Сформированы выводы и гипотезы для дальнейшего моделирования. | Python, pandas, numpy, matplotlib, seaborn |
| Statistical Analysis Project | Применены методы описательной статистики и статистического вывода. Проверка гипотез, анализ корреляций и интерпретация статистических результатов. | Python, pandas, scipy, statsmodels |
| Linear Model Project | Построены линейные модели для прогнозирования целевой переменной. Выполнена интерпретация коэффициентов и оценка качества моделей. | Python, pandas, scikit-learn |
| Linear Model in Oil Industry | Регрессионная модель для прикладной задачи из нефтяной отрасли. Анализ факторов, влияющих на целевой показатель, и оценка экономической интерпретации результатов. | Python, pandas, scikit-learn |
| Machine Learning Models | Реализация и сравнение нескольких алгоритмов машинного обучения. Настройка гиперпараметров и оценка качества моделей. | Python, pandas, scikit-learn |
| Gradient Boosting Models | Построение и анализ моделей градиентного бустинга. Сравнение качества с базовыми моделями, анализ важности признаков. | Python, pandas, scikit-learn, Gradient Boosting |
| Pipeline Implementation | Реализация полного ML-пайплайна: препроцессинг данных, кодирование признаков, масштабирование и обучение модели. | Python, pandas, scikit-learn |
| Pipeline Model for Internet Provider | Построение ML-пайплайна для задачи прогнозирования оттока клиентов интернет-провайдера. Оценка качества модели и интерпретация результатов. | Python, pandas, scikit-learn |
| Time Series Project | Анализ временных рядов: выявление трендов и сезонности, построение прогнозных моделей и оценка их качества. | Python, pandas, statsmodels |
| Natural Language Processing (NLP) | Обработка и анализ текстовых данных. Векторизация текста и построение моделей для задач классификации или анализа текста. | Python, nltk / sklearn, pandas |
| Computer Vision | Работа с изображениями: базовая обработка, извлечение признаков и применение моделей компьютерного зрения. | Python, OpenCV, numpy |
